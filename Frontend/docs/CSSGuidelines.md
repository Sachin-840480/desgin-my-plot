
Separate philosophy from CSS.

Example:

# CSS Guidelines

Use CSS to solve layout problems.

Not visual problems.

---

Prefer:

- gap
- padding
- margin
- border
- line-height
- font-size
- max-width

before using:

- transform
- filter
- backdrop-filter
- animation
- absolute positioning

---

## Shadows

Allowed.

Only when communicating elevation.

---

## Animation

Allowed.

Only when communicating state.

Never decorative.

---

## Radius

Buttons

4–6px

Inputs

4–6px

Cards

6–8px

Avoid pills unless the component is actually a pill.

---

## Containers

1080–1200px

Readable text

60–75 characters

---

Use

8

12
