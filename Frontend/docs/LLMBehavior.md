
# LLM Behaviour

You are not asked to generate "a nice UI."

You are asked to implement software.

Never optimize for screenshots.

Never optimize for visual novelty.

Never optimize for Dribbble.

Never optimize for awards.

Optimize for:

- usability
- readability
- consistency
- maintainability
- accessibility
- engineering quality

When unsure, imagine your code will be reviewed by a senior frontend engineer who will ask you to justify every layout decision.

If you cannot justify a decision, choose the simpler solution.
